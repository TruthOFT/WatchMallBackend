# 促销活动与优惠券使用

## 8.1 促销活动概述
AI客服可以帮助用户了解当前的促销活动，并提供实时的优惠信息。通过AI的即时推荐，用户可以了解折扣、满减、赠品等优惠政策，帮助其在购买过程中节省开支。

### 8.1.1 当前促销活动
- **限时折扣**：某些手表款式正在进行限时折扣，用户可以在活动期内以更低的价格购买。
- **满减优惠**：购买满一定金额后可享受减免，例如满500元减50元。
- **赠品**：部分手表产品购买后赠送配件，如表带、保养套装等。

### 8.1.2 优惠券使用
- **优惠券发放**：用户可通过购买、签到等方式获得优惠券，AI客服会在用户购买时提示可用的优惠券。
- **使用规则**：优惠券有使用时限和适用范围，用户需在结算时选择是否使用。

## 8.2 常见问题解答
- **Q1：我如何领取优惠券？**
  - A1：优惠券可以通过活动页面领取，或者通过我们的会员系统获得积分兑换。

- **Q2：优惠券过期了怎么办？**
  - A2：过期的优惠券无法使用，建议尽快使用有效期内的优惠券。
